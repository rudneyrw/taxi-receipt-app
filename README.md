# Aruba Taxi Driver – Full Production Vercel Version

## Included
- `index.html` → full driver app
- `receipt/index.html` → customer receipt page
- `manifest.json` → installable web app support
- `vercel.json` → Vercel config

## Features
- Default currency USD
- Fixed or manual locations
- Waiting time in 5-minute blocks at $5 per block
- Base fare + waiting charge + total fare
- Print receipt
- Customer QR receipt
- Hosted receipt link
- WhatsApp receipt

## Deploy
Import this repo into Vercel and deploy.
